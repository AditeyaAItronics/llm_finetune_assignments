S1 · Activations

Activations exist for a reason.

A model with no nonlinearity can only draw a straight boundary, so it cannot separate two interleaved rings. Same data, same training — the only change is the activation. Watch the linear model stall near 55% while one ReLU layer wraps the ring to ~99%.

S2 · Depth

Depth without nonlinearity is a lie.

Five stacked linear layers collapse to a single linear map, so a 5-layer linear net is no stronger than 1 layer — both fail the ring task identically. Insert ReLUs between the very same layers and it suddenly solves it.

The proof that nails it: multiply the five weight matrices of the linear stack together. Their product is a single 2×2-shaped map — algebraically identical to one layer.
depth_5_relu.py
# W5·W4·W3·W2·W1 is just one matrix. ReLU is what breaks the collapse.
linear5 = nn.Sequential(*[nn.Linear(2,2) for _ in range(5)])         # collapses
relu5   = nn.Sequential(*[layer for L in Ls for layer in (L, nn.ReLU())])  # does not
W = reduce(torch.matmul, [L.weight for L in linear5])       # == a single layer
S3 · Embeddings

Embeddings learn similarity from nothing but next-token.

A tiny model trained only to predict the next token — never told which words are related — makes the embedding table cluster related tokens on its own. Similarity was never supplied. The geometry of meaning emerges from next-token prediction alone.

S4 · Generalization

Memorization vs generalization — and data closes the gap.

A high-capacity model on tiny data drives train accuracy to ~100% while held-out accuracy stays low — it memorized. Grow the dataset and the gap collapses. This is the course's whole point: data is everything.
