# ERA V5 Assignments

My work for **ERA V5 — Extensive & Reimagined AI Program** by The School of AI.

## About the course

A lab-run, hands-on program where students collaborate to build, train, and release a frontier-scale Mixture-of-Experts language model — 20 weekly sessions, ~6 months, plus ongoing training-lab operations after the formal calendar ends.
